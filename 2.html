LESSON 05

الفقرات والنصوص

p · br · strong · em · mark · small · del · ins · sub · sup

📖
الفقرة <p> والسطر الجديد <br>

<p> هو وسم الفقرة — يضع النص في كتلة مستقلة مع مسافة تلقائية فوق وأسفل.

<br> يكسر السطر داخل نفس الفقرة (ذاتي الإغلاق — لا يحتاج </br>).

HTML 
<p>هذه فقرة كاملة. المتصفح يضيف مسافة تلقائية قبلها وبعدها.</p>

<p>السطر الأول<br>
السطر الثاني — داخل نفس الفقرة</p>
🎨
وسوم تنسيق النص

HTML يوفر وسوماً لتنسيق النص بطرق مختلفة دون الحاجة لـ CSS:

النتيجة في المتصفح
strong نص غامق مهم دلالياً
em نص مائل للتأكيد
mark نص مظلل بالأصفر
small نص صغير — للتنبيهات والحقوق
del نص محذوف (خط في المنتصف)
ins نص مُضاف (خط تحته)
b غامق بصري فقط (بلا معنى دلالي)
i مائل بصري فقط (بلا معنى دلالي)
HTML — وسوم النص 
<p>
  هذا نص <strong>مهم جداً</strong> وهذا <em>للتأكيد</em>
</p>
<p>
  السعر القديم: <del>100 ريال</del>
  السعر الجديد: <mark>75 ريال</mark>
</p>
💡
strong vs b: الفرق أن <strong> يعني "هذا النص مهم دلالياً" — محركات البحث وقارئات الشاشة تفهمه. أما <b> فهو تغمييق بصري فقط.
✏️
تمرين
1
صفحة إعلان منتج
  1. أنشئ صفحة لإعلان منتج
  2. أضف اسم المنتج بوسم strong
  3. أضف السعر القديم بـ del والسعر الجديد بـ mark
  4. أضف ملاحظة صغيرة بوسم small
الحل 
<p>منتج: <strong>حقيبة جلدية فاخرة</strong></p>
<p>
  السعر: <del>500 ريال</del>
  العرض: <mark>350 ريال</mark>
</p>
<p><small>* العرض ساري حتى نهاية الشهر</small></p>

// ملخص الدرس

  • <p> للفقرات، <br> لكسر السطر
  • <strong> للنص المهم دلالياً، <em> للتأكيد
  • <mark> للتظليل، <del> للنص المحذوف، <ins> للمضاف
  • <small> للنص الصغير مثل التنبيهات والحقوق
  • strong و em أفضل من b و i لأسباب SEO وإمكانية الوصول
LESSON 06

الروابط — <a>

href · target · title — الانتقال بين الصفحات والمواقع

📖
ما هو وسم <a>؟

وسم <a> (من Anchor = مرساة) يُنشئ رابطاً قابلاً للنقر. هو العنصر الأساسي للتنقل في الويب.

الخاصية الأهم هي href وهي اختصار لـ Hypertext Reference — تحدد وجهة الرابط.

HTML — الشكل الأساسي 
<a href="https://www.google.com">اذهب إلى جوجل</a>
⚙️
أنواع الروابط
🌐
رابط خارجي
ينقلك لموقع آخر على الإنترنت
https://...
📄
رابط داخلي
ينقلك لصفحة في نفس الموقع
about.html
#️⃣
رابط قسم
ينقلك لقسم في نفس الصفحة
#section-id
📧
رابط إيميل
يفتح برنامج البريد
mailto:...
HTML — جميع أنواع الروابط 
<!-- رابط خارجي -->
<a href="https://www.google.com">جوجل</a>

<!-- رابط لصفحة داخلية -->
<a href="about.html">من نحن</a>

<!-- رابط لقسم في نفس الصفحة -->
<a href="#contact">اذهب لقسم التواصل</a>

<!-- رابط إيميل -->
<a href="mailto:hello@example.com">راسلني</a>

<!-- رابط هاتف -->
<a href="tel:+966500000000">اتصل بنا</a>
⚙️
خاصية target

خاصية target تحدد أين يُفتح الرابط:

القيمةالوظيفة
_selfيفتح في نفس التبويب (الافتراضي)
_blankيفتح في تبويب جديد
_parentيفتح في الإطار الأب
_topيفتح في النافذة الكاملة
HTML — target + title 
<!-- يفتح في تبويب جديد -->
<a
  href="https://www.youtube.com"
  target="_blank"
  title="افتح يوتيوب في تبويب جديد"
>يوتيوب</a>
⚠️
عند استخدام target="_blank" أضف دائماً rel="noopener noreferrer" لأسباب أمنية:
<a href="..." target="_blank" rel="noopener noreferrer">
✏️
تمرين
1
صفحة تواصل
  1. أنشئ صفحة تحتوي على 4 روابط مختلفة
  2. رابط لموقع خارجي يفتح في تبويب جديد
  3. رابط لصفحة about.html داخلية
  4. رابط إيميل ورابط هاتف
الحل 
<body>
  <a href="https://google.com" target="_blank" rel="noopener noreferrer">
    جوجل — يفتح في تبويب جديد
  </a>

  <a href="about.html">من نحن</a>

  <a href="mailto:info@mysite.com">راسلنا على الإيميل</a>

  <a href="tel:+966500000000">اتصل بنا</a>
</body>

// ملخص الدرس

  • <a href="..."> ينشئ رابطاً — href هو وجهة الرابط
  • روابط خارجية تبدأ بـ https:// وداخلية باسم الملف
  • target="_blank" يفتح في تبويب جديد
  • mailto: للإيميل، tel: للهاتف
  • أضف rel="noopener noreferrer" مع target="_blank" دائماً
LESSON 07

الصور — <img>

src · alt · width · height · loading — إضافة الصور لصفحتك

📖
وسم <img>

<img> هو وسم ذاتي الإغلاق يعرض الصور. له خاصيتان أساسيتان:

src (Source) — مسار الصورة سواء كان ملفاً محلياً أو رابط إنترنت.

alt (Alternative Text) — نص بديل يظهر إذا تعذّر تحميل الصورة، ومهم جداً للـ SEO وإمكانية الوصول.

HTML — الشكل الأساسي 
<!-- صورة من ملف محلي -->
<img src="images/photo.jpg" alt="صورة شخصية">

<!-- صورة من الإنترنت -->
<img src="https://example.com/image.png" alt="وصف الصورة">
⚙️
الخصائص المهمة
الخاصيةالوظيفةمثال
src مسار الصورة — مطلوب دائماً src="photo.jpg"
alt نص بديل — مطلوب دائماً alt="صورة شعار الموقع"
width عرض الصورة بالـ px أو % width="300"
height ارتفاع الصورة height="200"
loading تحميل كسول — لتحسين الأداء loading="lazy"
title نص يظهر عند تمرير الماوس title="اضغط لتكبير الصورة"
HTML — صورة كاملة الخصائص 
<img
  src="images/hero.jpg"
  alt="منظر طبيعي جميل عند غروب الشمس"
  width="800"
  height="400"
  loading="lazy"
  title="صورة غروب الشمس"
>
🔍
alt مهم جداً: لا تتركه فارغاً أبداً (إلا إذا كانت الصورة زخرفية بحتة). وصف جيد لـ alt يساعد محركات البحث ويجعل موقعك أكثر شمولاً لذوي الإعاقات البصرية.
💡
رابط صورة — <a> داخل <a>

يمكنك جعل الصورة نفسها رابطاً بتغليفها بوسم <a>:

HTML — صورة قابلة للنقر 
<a href="https://www.example.com">
  <img src="logo.png" alt="شعار موقعنا — اضغط للرجوع للرئيسية">
</a>
✏️
تمرين
1
معرض صور بسيط
  1. أنشئ مجلداً باسم images وضع فيه 3 صور
  2. في index.html أضف 3 عناصر <img> مع alt صحيح لكل واحدة
  3. حدد عرض الصور بـ width="300"
  4. اجعل إحدى الصور رابطاً ينقل لصفحة about.html
الحل 
<img src="images/photo1.jpg" alt="صورة الطبيعة" width="300">
<img src="images/photo2.jpg" alt="صورة المدينة" width="300">

<!-- الصورة الثالثة كرابط -->
<a href="about.html">
  <img src="images/photo3.jpg" alt="اضغط لصفحة من نحن" width="300">
</a>

// ملخص الدرس

  • <img> وسم ذاتي الإغلاق لعرض الصور
  • src يحدد مسار الصورة — مطلوب دائماً
  • alt نص بديل — مطلوب دائماً ومهم للـ SEO
  • width و height لتحديد أبعاد الصورة
  • loading="lazy" يحسن أداء الصفحة
LESSON 08

الجداول — table

table · tr · th · td · thead · tbody · colspan · rowspan

📖
هيكل الجدول

جدول HTML يتكون من صفوف وأعمدة. الوسوم الأساسية:

الوسمالوظيفة
<table>الحاوية الرئيسية للجدول
<thead>قسم رأس الجدول (اختياري لكن مستحسن)
<tbody>قسم جسم الجدول
<tr>Table Row — صف كامل
<th>Table Header — خلية عنوان (غامق ومتوسط)
<td>Table Data — خلية بيانات عادية
💻
مثال جدول كامل
HTML — جدول طلاب 
<table border="1">
  <thead>
    <tr>
      <th>الاسم</th>
      <th>العمر</th>
      <th>المدينة</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>أحمد</td>
      <td>22</td>
      <td>الرياض</td>
    </tr>
    <tr>
      <td>سارة</td>
      <td>25</td>
      <td>جدة</td>
    </tr>
  </tbody>
</table>
النتيجة في المتصفح
الاسمالعمرالمدينة
أحمد22الرياض
سارة25جدة
محمد20الدمام
⚙️
دمج الخلايا: colspan و rowspan

colspan يدمج خلايا أفقياً (عبر أعمدة). rowspan يدمج خلايا عمودياً (عبر صفوف).

HTML — colspan و rowspan 
<table border="1">
  <tr>
    <!-- تمتد هذه الخلية عبر عمودين -->
    <td colspan="2">خلية تمتد عمودين</td>
    <td>خلية عادية</td>
  </tr>
  <tr>
    <!-- تمتد هذه الخلية عبر صفين -->
    <td rowspan="2">خلية تمتد صفين</td>
    <td>A</td>
    <td>B</td>
  </tr>
</table>
✏️
تمرين
1
جدول جدول دراسي
  1. أنشئ جدولاً يحتوي على 3 أعمدة: المادة، المعلم، الدرجة
  2. أضف 4 صفوف من البيانات
  3. استخدم thead و tbody بشكل صحيح
  4. اجعل عنوان "جدولي الدراسي" يمتد فوق الجدول كله بـ colspan
الحل 
<table border="1">
  <thead>
    <tr>
      <th colspan="3">جدولي الدراسي</th>
    </tr>
    <tr>
      <th>المادة</th><th>المعلم</th><th>الدرجة</th>
    </tr>
  </thead>
  <tbody>
    <tr><td>رياضيات</td><td>أ. خالد</td><td>95</td></tr>
    <tr><td>فيزياء</td><td>أ. سارة</td><td>88</td></tr>
    <tr><td>HTML</td><td>أ. محمد</td><td>100</td></tr>
  </tbody>
</table>

// ملخص الدرس

  • <table> الحاوية، <tr> صف، <th> عنوان خلية، <td> بيانات
  • <thead> لرأس الجدول، <tbody> للبيانات
  • colspan يدمج الخلايا أفقياً، rowspan عمودياً
  • الجداول للبيانات فقط — لا تستخدمها للتصميم
LESSON 09

القوائم — ul · ol · dl

قوائم غير مرتبة، مرتبة، ووصفية

📖
أنواع القوائم الثلاثة
<ul> — غير مرتبة
  • HTML
  • CSS
  • JavaScript
<ol> — مرتبة
  1. تثبيت VS Code
  2. كتابة HTML
  3. رفع الموقع
<dl> — وصفية
HTML
لغة الهيكل
CSS
لغة التصميم
💻
الكود
HTML — القوائم الثلاثة 
<!-- قائمة غير مرتبة (نقاط) -->
<ul>
  <li>HTML</li>
  <li>CSS</li>
  <li>JavaScript</li>
</ul>

<!-- قائمة مرتبة (أرقام) -->
<ol>
  <li>تثبيت VS Code</li>
  <li>كتابة الكود</li>
  <li>رفع الموقع</li>
</ol>

<!-- قائمة وصفية -->
<dl>
  <dt>HTML</dt>
  <dd>لغة هيكل صفحات الويب</dd>
  <dt>CSS</dt>
  <dd>لغة تصميم وتنسيق الصفحات</dd>
</dl>
💡
قوائم متداخلة (Nested Lists)

يمكنك وضع قائمة داخل عنصر <li> لإنشاء قوائم متداخلة:

HTML — قوائم متداخلة 
<ul>
  <li>تطوير الويب
    <ul>
      <li>HTML</li>
      <li>CSS</li>
      <li>JavaScript</li>
    </ul>
  </li>
  <li>تطوير الموبايل</li>
</ul>
✏️
تمرين
1
صفحة وصفة طبخ
  1. أنشئ صفحة لوصفة طعام
  2. استخدم <ul> لقائمة المكونات
  3. استخدم <ol> لخطوات التحضير بالترتيب
  4. استخدم <dl> لقيم التغذية (سعرات، بروتين...)
الحل 
<h1>وصفة الشوربة</h1>

<h2>المكونات</h2>
<ul>
  <li>دجاج</li>
  <li>جزر</li>
  <li>بصل</li>
</ul>

<h2>خطوات التحضير</h2>
<ol>
  <li>قطّع الدجاج</li>
  <li>سخّن الزيت</li>
  <li>أضف المكونات وانتظر 30 دقيقة</li>
</ol>

<h2>القيم الغذائية</h2>
<dl>
  <dt>السعرات</dt>
  <dd>250 كيلوكالوري</dd>
  <dt>البروتين</dt>
  <dd>30 جرام</dd>
</dl>

// ملخص الدرس

  • <ul> قائمة بنقاط (غير مرتبة)، <li> لكل عنصر
  • <ol> قائمة بأرقام (مرتبة)
  • <dl> قائمة وصفية — <dt> للمصطلح، <dd> للتعريف
  • القوائم يمكن تداخلها — li داخل li
LESSON 10

الصوت — <audio>

تضمين ملفات صوتية في صفحتك مع التحكم الكامل

📖
وسم <audio>

وسم <audio> يُضمّن ملفات صوتية مباشرة في الصفحة. يدعم المتصفحون الحديثون صيغ MP3 و OGG و WAV.

HTML — الشكل الأساسي 
<audio controls>
  <source src="music.mp3" type="audio/mpeg">
  <source src="music.ogg" type="audio/ogg">
  <!-- إذا المتصفح لا يدعم audio -->
  متصفحك لا يدعم تشغيل الصوت.
</audio>
💡
استخدام <source> متعدد يضمن التوافق — المتصفح يختار أول صيغة يدعمها.
⚙️
خصائص <audio>
controls يظهر أزرار التحكم (تشغيل، إيقاف، مستوى الصوت)
autoplay يبدأ التشغيل تلقائياً عند تحميل الصفحة
loop يكرر التشغيل بشكل مستمر
muted يبدأ الصوت مكتوماً
preload كيفية التحميل المسبق: auto / metadata / none
HTML — audio مع جميع الخصائص 
<!-- صوت مع تحكم كامل -->
<audio
  controls
  loop
  preload="metadata"
>
  <source src="audio/podcast.mp3" type="audio/mpeg">
  متصفحك لا يدعم الصوت.
</audio>
⚠️
تجنب استخدام autoplay دون muted — معظم المتصفحات تمنعه الآن لأنه مزعج للمستخدم.

// ملخص الدرس

  • <audio> لتضمين الصوت في الصفحة
  • <source> داخله لتحديد الملف والنوع
  • controls يظهر أزرار التشغيل للمستخدم
  • loop للتكرار، muted للبدء صامتاً
  • أضف نصاً بديلاً داخل audio للمتصفحات القديمة
LESSON 11

الفيديو — <video>

تضمين فيديوهات محلية مع التحكم الكامل في خصائص العرض

📖
وسم <video>

وسم <video> يُضمّن ملفات فيديو مباشرة. يدعم الصيغ: MP4 (الأكثر توافقاً) و WebM و OGG.

HTML — الشكل الأساسي 
<video controls width="640" height="360">
  <source src="video.mp4" type="video/mp4">
  <source src="video.webm" type="video/webm">
  متصفحك لا يدعم تشغيل الفيديو.
</video>
⚙️
خصائص <video>
controls يظهر شريط التحكم (تشغيل، ملء شاشة، صوت...)
autoplay يشغّل الفيديو تلقائياً (يشترط muted غالباً)
loop يكرر الفيديو تلقائياً
muted يبدأ الفيديو بدون صوت
poster صورة تظهر قبل التشغيل (thumbnail)
width / height أبعاد عرض الفيديو
playsinline يشغّل داخل الصفحة على iOS (لا يفتح fullscreen)
preload auto / metadata / none
HTML — فيديو خلفية صامت + فيديو عادي بـ poster 
<!-- فيديو خلفية — شائع في الـ hero sections -->
<video
  autoplay
  muted
  loop
  playsinline
>
  <source src="hero-bg.mp4" type="video/mp4">
</video>

<!-- فيديو عادي مع صورة انتظار -->
<video
  controls
  width="800"
  poster="thumbnail.jpg"
  preload="metadata"
>
  <source src="lesson.mp4" type="video/mp4">
  متصفحك لا يدعم الفيديو.
</video>
💡
audio vs video — مقارنة سريعة
النقطة<audio><video>
الصيغMP3, OGG, WAVMP4, WebM, OGG
posterغير موجود✓ صورة انتظار
الأبعادلا يحتاجwidth و height
playsinlineغير ضروري✓ للـ iOS
مشتركcontrols · autoplay · loop · muted · preload · <source>
✏️
تمرين
1
صفحة ميديا كاملة
  1. أنشئ صفحة تحتوي على <audio> و <video> معاً
  2. للصوت: أضف controls و loop
  3. للفيديو: أضف controls و poster و width="600"
  4. أضف نصاً بديلاً داخل كليهما للمتصفحات القديمة
الحل 
<h2>موسيقى الخلفية</h2>
<audio controls loop>
  <source src="music.mp3" type="audio/mpeg">
  متصفحك لا يدعم الصوت.
</audio>

<h2>فيديو الدرس</h2>
<video
  controls
  width="600"
